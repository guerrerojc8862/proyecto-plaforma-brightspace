La función input()

La función input() permite obtener datos desde la consola, los cuales son ingresados por el usuario. Estos datos son guardados con tipo string


Ejemplo:

                                name = input("Ingresa tu nombre: ")
                                print("Hola, " + name + ". ¡Un gusto conocerte!")

Conversión de datos (casting)

Dado que la función input() nos guarda datos de tipo string, esto puede generarnos problemas si lo que deseamos es operar con un numero ingresado por el usuario. Python ofrece dos simples funciones para especificar un tipo de dato y resolver este problema, aquí están: int() y float().


Sus nombres indican cual es su función:

  • La función int() toma un argumento (por ejemplo, una cadena: int(string)) e intenta convertirlo a un valor entero; si llegase a fallar, el programa entero fallará también.

  • La función float() toma un argumento (por ejemplo, una cadena: float(string)) e intenta convertirlo a flotante.

Cuando la función input() es llamada o invocada, el flujo del programa se detiene, el símbolo del cursor se mantiene parpadeando (le está indicando al usuario que tome acción ya que la consola está en modo de entrada) hasta que el usuario haya ingresado un dato y/o haya presionado la tecla Enter.
