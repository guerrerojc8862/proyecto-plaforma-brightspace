Los Bucles (Ciclos) en Python

En Python existen dos tipos de Bucles: while y for:

  • El bucle while ejecutar una sentencia o un conjunto de sentencias siempre que una condición booleana especificada sea verdadera:

                                            while condition_true:
                                            code_to_execute

  • El bucle for ejecuta un conjunto de sentencias muchas veces; se usa para iterar sobre una secuencia (por ejemplo, una lista, un diccionario, una tupla o un conjunto; pronto aprenderás sobre ellos) u otros objetos que son iterables (por ejemplo, cadenas). Puedes usar el bucle for para iterar sobre una secuencia de números usando la función incorporada range. 

                                            for element in iterable_object:
                                            code_to_execute

Las sentencias break y continue

  • Puedes usar la sentenica break para terminar la iteración dentro de un bucle

                                            for element in iterable_object:
                                            code_to_execute
                                            break

  • Puedes usar la sentenica continue para continuar con la siguiente iteración dentro de un bucle

                                            for element in iterable_object:
                                            code_to_execute
                                            continue

  • La función range() genera una secuencia de números. Acepta enteros y devuelve objetos de rango. La sintaxis de range() tiene el siguiente aspecto range(start,stop,step), donde:

    • start es un parámetro opcional que especifica el inicio de la secuencia
    • stop es un parámetro opcional que especifica el final de la secuencia (no incluido el valor)
    • step es un parámetro que especifica el incremento en la secuencia
