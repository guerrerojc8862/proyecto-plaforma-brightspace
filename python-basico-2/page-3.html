Operadores Lógicos en Python

Los operadores lógicos nos permiten evaluar el grado de verdad de una o más sentencias lógicas. Los operadores lógicos en Python son and, or, not

  • and o el operador conjunción determina el grado de verdad de dos sentencias, y solo es cierto si ambas sentencias son ciertas.
    P Q P and Q
    true true true
    true false false
    false true false
    false false false
  • or o el operador disyunción determina el grado de verdad de dos sentencias, y es cierto siempre y cuando al menos una sentencia sea cierta.
    P Q P and Q
    true true true
    true false true
    false true true
    false false false

  • not o el operador negación invierte el valor de verdad de una sentencia

    P not P
    true false
    false true
